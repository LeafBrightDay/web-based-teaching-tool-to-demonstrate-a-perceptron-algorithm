Biological Neural Networks

The basic theory of BNNs is that a neuron will be fired if it has received from the presynaptic neurons a summary impulse, which exceeds a particular threshold. Each impulse is all-or-none event but the number of ionic channels triggered by the signal is different from synapse to synapse. The signal from a single synapse is able to overcome the threshold and force a neuron to fire an action potential, but other synapses could realise this only by simultaneously delivering their signals. Therefore, input from every synapse, or "connection", to the neuron in the model network must be assigned with some value w, called weight of connection. Given that, it is able to draw an abstract "neuron" as a model unit of the artificial neural network.

User Guide

The web applet is designed for visually demonstrating the procedure of perceptron learning rule. To use it, please reading the following instructions.

Step 1: Click the tab 'Run' to start the applet.

Step 2: Please enter valid numbers of learning rate (C), stop value (stopValue), threshold and maximum number of iterations(maximum_iteration). Typically, the range of C is 0.1~0.5, the stopValue is <= 0.01, the length of threshold is 3 and the maximun_iteration could be <=100 (training set here is small, no need for large amount of iterations).

Step 3: Just click the "Define training sets" button to define the training set which has 4 patterns.

Step 4: If you hope to redefine the training sets, you could click "Modify training sets" button and choose the particular pattern to change its value.

Step 5: Ensure above settings are ok, please click "Start" button to begin the simulation.

Step 6: Click "Next" button to observe the update of weight connections and output values during each iteration.

Step 7: If the perceptron is successfully trained, "Enter testing set" button is okey for defining a new testing set to check if the perceptron is able to recognize a pattern.

Note: If you find any problem in the applet, you can click the "Guide" button to see the instructions.

Warning: Any incorrect use of the applet will lead to a refresh (F5) of the web page in order to obtain a correct learning experience.
